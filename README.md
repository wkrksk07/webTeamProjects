# 홍익대학교 세종캠퍼스 3D 맵 프로젝트

본 프로젝트는 홍익대학교 세종캠퍼스 건물을 3D로 시각화하여 강의실 위치 안내 및 학내 편의 정보를 제공하는 웹 서비스입니다.

## 🚀 주요 기능
- **다층 3D 시각화**: FBX 모델링을 활용한 2F~4F 층별 구조 확인.
- **역동적인 애니메이션**: 층별 선택 시 해당 층 외의 모델이 날아가는(Fly-out) 효과 및 부드러운 카메라 전환.
- **인터랙티브 핀 시스템**: 얇은 다리와 빨간 공 디자인의 마커를 통해 강의실 위치 표시 및 포커싱.
- **실시간 정보 연동**: 학교 식당 웹사이트와 연동된 오늘/내일 식단 정보 제공 (시설 안내 페이지).
- **좌표 수집 도구**: 개발자 모드에서 모델 클릭 시 실시간 좌표 추출 가능.

## 🛠 기술 스택
- **프론트엔드**: Vanilla JavaScript, [Three.js](https://threejs.org/)
- **빌드 도구**: [Vite](https://vitejs.dev/)
- **모델링**: FBX (A동 2~4층)
- **크롤링**: Cheerio 기반 식단 정보 추출 (구현 완료)

## 📁 프로젝트 구조
```text
/
├── GEMINI.md               # 상세 개발 지침 및 로드맵
├── README.md               # 프로젝트 개요 및 기능 설명
└── webSite/                # 웹 소스 코드
    ├── facility.html       # 시설 안내 및 식단 정보 페이지
    ├── index.html          # 메인 3D 맵 페이지
    ├── CRAWLING_GUIDE.md   # 식단 크롤링 관련 기술 가이드
    ├── public/
    │   ├── Img/            # 학교 로고 및 이미지 리소스
    │   └── models/         # FBX 3D 모델 파일 (2F, 3F, 4F)
    └── src/
        ├── main.js         # Three.js 씬 구성 및 핵심 로직
        └── style.css       # 웹 UI 및 레이아웃 스타일
```

## 💻 시작하기
1. `webSite` 폴더로 이동: `cd webSite`
2. 의존성 설치: `npm install`
3. 개발 서버 실행: `npm run dev`
