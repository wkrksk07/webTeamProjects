# 홍익대학교 세종캠퍼스 3D 맵 정보 크롤링 가이드

본 문서는 학교 웹사이트의 정보를 크롤링하여 3D 맵 서비스와 연동하는 방법 및 주의사항을 기술합니다.

## 1. 대상 데이터 후보
- **공지사항**: 학교 홈페이지 메인 또는 학과별 공지 게시판.
- **시설 정보**: 캠퍼스 가이드 내 건물별 부서 및 연락처.
- **학식 정보**: 학생 식당의 일자별 메뉴 데이터.

## 2. 기술 스택 추천
- **Language**: Python (가장 범용적)
- **Libraries**:
  - `requests`: 웹 페이지 HTML 요청.
  - `BeautifulSoup4`: HTML 파싱 및 데이터 추출.
  - `Selenium` / `Playwright`: 동적 콘텐츠(자바스크립트 실행 필요 시) 크롤링.

## 3. 기본 아키텍처
1. **Collector**: Python 스크립트가 주기적으로 학교 사이트 방문.
2. **Storage**: 추출된 데이터를 `data.json` 또는 Database에 저장.
3. **API/Frontend**: 3D 맵 웹사이트에서 `fetch('/path/to/data.json')`를 통해 화면에 표시.

## 4. 주의사항
- **robots.txt**: 학교 홈페이지의 크롤링 허용 정책을 반드시 확인하십시오.
- **Traffic**: 과도한 요청은 서버 차단의 원인이 됩니다. (최소 1시간 이상의 갱신 주기 권장)
